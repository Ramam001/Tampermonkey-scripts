# Tampermonkey Scripts

A collection of user scripts designed to enhance and modify web browsing experiences using [Tampermonkey](https://www.tampermonkey.net/).

## 📋 Scripts

### Gemini Cursive Handwriting Scholar V3
A script that enhances the Google Gemini interface with cursive handwriting analysis capabilities. Allows you to submit handwritten text for scholarly evaluation and feedback.

### Instagram Profile Media Filter - Reels
Filters and organizes Instagram profile media, with specific focus on reels. Customize your Instagram browsing experience by filtering content types on user profiles.

## 🚀 Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) browser extension for your browser
2. Navigate to the script files in this repository
3. Click the "Raw" button to view the script code
4. Tampermonkey will prompt you to install the script
5. Click "Install" and you're done!

## 📖 Usage

- Each script activates automatically on its target website(s)
- Functionality is designed to be non-intrusive and work seamlessly with the original site
- Check the script headers for specific website requirements and configurations

## ⚙️ Requirements

- Tampermonkey browser extension installed
- Target websites must be accessible
